# 🍣 かっぱ寿司 食べホー元取り計算機 (Kappa Sushi AYCE Calculator)

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Deployed-success)](#)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

かっぱ寿司の食べ放題プラン（食べホー）で、「どれだけ元を取れたか」をゲーム感覚でリアルタイムに計算できるWebアプリケーションです。

👉 **[アプリを実際に動かしてみる](https://masaya-aoy.github.io/kappasushi-tabeho-/)**

## 📱 概要 (Overview)
食事中にスマホから手軽に入力できるよう、片手での操作性に特化したUIを採用しています。メニューごとに食べた数をカウントダウン/アップしていくことで、現在の合計金額と目標金額（食べ放題の料金）までの差額が一目でわかります。

## ✨ 主な機能 (Features)
- **スマホ最適化UI**: タップしやすい大きなボタンと、スクロールしても常に合計金額が見える固定ヘッダー。
- **カテゴリ別タブ＆ページネーション**: 膨大なメニューから目的の寿司をすぐに見つけられるよう、カテゴリごとにタブ分けし、1画面5件ずつのページネーションを実装。
- **達成感の演出**: 目標金額に近づくにつれてヘッダーの背景色が白から黄色へグラデーション変化。元を取った瞬間に祝福のメッセージが表示されます。
- **ヘッドレスCMS的な運用**: メニューデータはGoogleスプレッドシートで管理。アプリのコードを改修することなく、店長や非エンジニアでもメニューの追加・価格変更が即座に反映可能です。

## 🛠 技術スタック (Tech Stack)
- **フロントエンド**: HTML5, Vanilla JavaScript, Tailwind CSS (CDN)
- **バックエンド (API)**: Google Apps Script (GAS)
- **データベース**: Google Spreadsheet
- **ホスティング**: GitHub Pages

## 🏗 アーキテクチャ (Architecture)
このアプリはフロントエンドとバックエンドを分離したモダンな構成で作られています。
1. GitHub Pages上の静的HTML(JavaScript)が読み込まれる。
2. JavaScriptの `fetch` APIを使用して、GASで構築した専用APIエンドポイントへGETリクエストを送信。
3. GASがGoogleスプレッドシートの最新メニュー情報をJSON形式で返す。
4. 受け取ったJSONデータを元に、フロントエンドでDOMを動的に生成しメニューを描画。

## 💡 使い方 (Usage)
1. 上部のタブからカテゴリ（にぎり、軍艦・巻物、サイドメニュー、スイーツ）を選択します。
2. 食べた商品の右側にある `+` ボタンをタップして数を増やします（間違えたら `-` で減らせます）。
3. 画面上部に「計〇〇円」「あと〇〇円で元が取れます！」とリアルタイムに表示されます。
4. 全てリセットしたい場合は、右上の「リセット」ボタンを押してください。
